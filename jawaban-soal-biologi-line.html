Penjelasan dengan langkah-langkah:

[tex]\tt{}5e + 20\% - 3! - 0.2[/tex]

[tex] \tt{}5e + \frac{20}{100} - 3! - \frac{1}{5} [/tex]

[tex] \tt{}5e + 0 - 6[/tex]

[tex]\tt{}5e - 6[/tex]

[tex]\tt{}7.59141[/tex]

quiz

4/7

dancing line the mathematics

soal ada di foto

<=> JAWABAN <=>

5e + 20% - 3! - 0.2 = 7,59141

<=> PEMBAHASAN <=>Langkah Menjawab :

Dengan Definisi p% = p/100Ubah bilangan desimal menjadi bentuk pecahanKurangkanlah pecahan-pecahan tersebutHitung faktorial tersebutMenghilangkan 0 tidak mengubah nilai, maka hilangkanlah dari pernyataan tersebutPenyelesaian

<=> Penyelesaian Soal :

5e + 20% - 3! - 0.2 =

= 5e + 20/100 - 3! - 0.2

= 5e + 20/100 - 3! - 1/5

= 20/100 - 1/5

= 1/5 - 1/5

= 0

= 5e + 0 - 3!

= 5e + 0 - 6

= 5e - 6

= 7,59141
