23. Mengapa HTML bukan termasuk bahasa pemrograman?


Jawaban:

HTML adalah singkatan dari Hypertext Markup Language. Pengertian HTML sering kali dikaitkan dengan bahasa pemrograman. Akan tetapi, sejatinya, HTML bukan sebuah bahasa pemrograman (programming languange), melainkan sebuah markup languange.

Jadi HTML yang merupakan bahasa markup tidak termasuk bahasa pemrograman karena tidak memiliki tag atau kode yang berfungi untuk menyelesaikan masalah. HTML hanya berfungsi memberikan tampilan luar bagi bahasa pemrograman web seperti php dan JavaScript, bersama dengan CSS.

39. PADA TAHUN BERAPA BAHASA PEMROGRAMAN HTML DI CIPTAKAN​


Jawaban:

Hypertext merujuk pada teks yang memuat referensi (link) ke teks lain yang bisa diakses langsung oleh viewer. Tim merilis versi pertama HTML pada tahun 1991, dan di dalamnya terdiri atas 18 HTML tag. Sejak saat itu, setiap kali bahasa HTML merilis versi teranyarnya, selalu ada tag dan attribute (tag modifier) terbaru.

Jawaban:

Sejarah HTML atau Hypertext Markup Language telah menjadi teknologi utama untuk Web sejak awal tahun 1990-an. Tim Berners-Lee menciptakan HTML pada tahun 1989 sebagai cara sederhana namun efektif untuk mengkodekan dokumen elektronik. Bahkan, tujuan awal dari web browser adalah untuk melayani pembaca untuk membuka dokumen berformat HTML.
