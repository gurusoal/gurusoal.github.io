23. Google drive adalah layanan google untuk.....

Digunakan sebagai tempta penyimpanan file baik itu file yang dimasukkan langsung atau file back up seperti file back up dari aplikasi WhatsApp.Digunakan sebagai saranan sharing file antara sesama. Kapasitas penyimpanan layanan versi gratis sudah pasti lebih sedikit jika dibandingkan dengan layanan versi berbayar. Di mana layanan versi gratis memberikan akses penyimpanan  maksimal hingga 15 GB dan layanan versi berbayar memberikan akses penyimpanan hingga 100 GB, 200 GB hingga 2 TB.
