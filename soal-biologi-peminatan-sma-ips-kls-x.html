Sinopsis

Buku ini disusun dan dirancang untuk mengembangkan sikap, pengetahuan, dan keterampilan siswa tentang berbagai fenomena sosial yang ada di masyarakat.

Fitur-fitur unggulan:
1. Nilai karakter berisi karakter yan gdapat dikembangkan setelah mempelajari materi yang dibahas pada suatu bab.
2. Peta Konsep,berisi sejumlah konsep penting yang akan dipelajari dan perlu dipahami siswa.

7. Glosarium dan Indeks, memperkenalkan konsep-konsep sosiologi yang memudahkan siswa menemukan pembahasan suatu konsep.

22. Sebutkan kelompok peminatan yang ada di SMA ?


Telah terdapat kelompok peminatan pada sebuah SMA yaitu ialah sebagai berikut:

Peminatan Matematika dan juga Sains ini sendiri telah meliputi mata pelajaran dari Matematika. Kimia, Fisika, dan juga Biologi. Peminatan Sosial yang di mana telah meliputi sebuah mata pelajaran ekonomi, sosiologi, geografi, antropologi, dan juga sejarah. Peminatan Bahasa yang di mana telah meliputi mata pelajaran dari Bahasa dan juga Sastra Indonesia, Bahasa dan juga Sastra lainnya, Bahasa dan juga Sastra Inggris, dan juga antropologi. Pembahasan

Pada seharusnya, peminatan ini sendiri tidak perlu untuk membuat pusing dikarenakan pada dasarnya sekolah ini telah mempunyai sebauh sistem yang dapat membantu untuk memiliki kelas IPS/IIS, IPA/MIA, ataupun Bahasa dan juga Budaya. Acuan dasar dari peminatan ini sendiri pada umumnya telah bergantung pada nilai rapot kita pada SMA ataupun MTS.

Apabila nilai mata pelajaran IPA ini lebih bagus dibandingkan dengan mata pelajaran dari IPS maka dari itu kita akan segera diarahkan untuk dapat masuk ke kelas IPA. Namun, kita masih terdapat peluang untuk masuk ke dalam kelas IPS tetapi kita harus dihadapkan terlebih dahulu dengan guru dari BP/BK.

Melakukan sebuah konsultasi dengan guru BP/BK telah dapat membantu kita untuk menemukan pada bagian mana sebenarnya minat dan juga kemampuan yang kita miliki ini.

37. tolong bantuin, soal matematika peminatan kls X semester 2


Q, R dan S adalah titik – titik pada garis yang menghubungkan titik P(a, x) dan T(b, y) sedemikian sehingga PQ = QR = RS = ST, maka [tex] (\frac{5a + 3b}{8}, \frac{5x + 3y}{8} )[/tex] adalah titik tengah dari ....
A. PQ
B. QR
C. RS
D. ST
E. tak satupun benar  

Pembahasan :  

R adalah titik tengah PT
R = [tex]( \frac{P + T}{2} )[/tex]
R = [tex]( \frac{(a, x) + (b, y)}{2} )[/tex]
R = [tex]( \frac{(a + b), (x + y)}{2} )[/tex]
R = [tex]( \frac{a + b}{2}, \frac{x + y}{2} )[/tex]

Q adalah titik tengah PR
Q = [tex]( \frac{P + R}{2} )[/tex]
Q = [tex]( \frac{(a, x) + ( \frac{a + b}{2}, \frac{x + y}{2} )}{2} )[/tex]
Q = [tex]( \frac{a + \frac{a + b}{2}, x + \frac{x + y}{2} }{2} )[/tex]
Q = [tex] ( \frac{ \frac{2a + a +b }{2}, \frac{2x + x + y}{2} }{2} )[/tex]
Q = [tex]( \frac{ 3a + b }{4}, \frac{3x + y}{4} )[/tex]

S adalah titik tengah RT
S = [tex]( \frac{R + T}{2} )[/tex]
S = [tex]( \frac{( \frac{a + b}{2}, \frac{x+y}{2} ) + (b, y)}{2} )[/tex]
S = [tex]( \frac{ \frac{a+b}{2}+b, \frac{x+y}{2}+y }{2} )[/tex]
S = [tex]( \frac{ \frac{a+b+2b}{2}, \frac{x+y+2y}{2} }{2} )[/tex]
S = [tex]( \frac{a+3b}{4}, \frac{x+3y}{4} )[/tex]

Titik tengah PQ
= [tex]( \frac{P+Q}{2} )[/tex]
= [tex]( \frac{(a,x)+( \frac{3a+b}{4}, \frac{3x+y}{4} )}{2} )[/tex]
= [tex]( \frac{a + \frac{3a+b}{4}, x + \frac{3x+y}{4} }{2} )[/tex]
= [tex]( \frac{ \frac{4a+3a+b}{4}, \frac{4x+3x+y}{4} }{2} )[/tex]
= [tex]( \frac{7a+b}{8}, \frac{7x+b}{8} )[/tex]

Titik tengah QR
= [tex]( \frac{Q+R}{2} )[/tex]
= [tex]( \frac{( \frac{3a+b}{4}, \frac{3x+y}{4} )+( \frac{a+b}{2}, \frac{x+y}{2} )}{2} )[/tex]
= [tex]( \frac{ \frac{3a+b}{4}+ \frac{a+b}{2}, \frac{3x+y}{4}+ \frac{x+y}{2} }{2} )[/tex]
= [tex]( \frac{ \frac{3a+b+2a+2b}{4}, \frac{3x+y+2x+2y}{4} }{2} )[/tex]
= [tex]( \frac{5a+3b}{8}, \frac{5x+3y}{8} )[/tex]

Titik tengah RS
= [tex]( \frac{R+S}{2} )[/tex]
= [tex]( \frac{( \frac{a+b}{2}, \frac{x+y}{2} )+( \frac{a+3b}{4}, \frac{x+3y}{4} )}{2} )[/tex]
= [tex]( \frac{ \frac{a+b}{2}+ \frac{a+3b}{4}, \frac{x+y}{2}+ \frac{x+3y}{4} }{2} )[/tex]
= [tex]( \frac{ \frac{2a+2b+a+3b}{4}, \frac{2x+2y+x+3y}{4} }{2} )[/tex]
= [tex]( \frac{3a+5b}{8}, \frac{3x+5y}{8} )[/tex]

Titik tengah ST
= [tex]( \frac{S+T}{2} )[/tex]
= [tex]( \frac{( \frac{a+3b}{4}, \frac{x+3y}{4} )+(b,y)}{2} )[/tex]
= [tex]( \frac{ \frac{a+3b}{4}+b, \frac{x+3y}{4}+y }{2} )[/tex]
= [tex]( \frac{ \frac{a + 3b+4b}{4}, \frac{x+3y+4y}{4} }{2} )[/tex]
= [tex]( \frac{a+7b}{8}, \frac{x+7y}{8} )[/tex]

Jadi 

[tex]( \frac{5a+3b}{8}, \frac{5x+3y}{8} )[/tex] adalah titik tengah dari QR

Jawaban : B. QR
