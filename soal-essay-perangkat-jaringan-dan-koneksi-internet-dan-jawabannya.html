18. 1. Menentukan kebutuhan perangkat keras jaringan yang akan digunakan, agar kedua laboratorium tersebut terkoneksi dengan internet.​


Untuk menghubungkan kedua laboratorium ke internet, kamu memerlukan perangkat keras jaringan yang mencakup:

1. Router: Router akan berfungsi sebagai gateway ke internet. kamu memerlukan setidaknya satu router untuk menghubungkan kedua laboratorium ke internet.

2. Switch: Switch digunakan untuk menghubungkan semua perangkat dalam laboratorium dan memungkinkan pertukaran data di dalam jaringan lokal.

3. Kabel Jaringan: kamu memerlukan kabel jaringan (ethernet) untuk menghubungkan semua perangkat (komputer, printer, dll.) ke switch dan router.

4. Modem: Jika kamu tidak memiliki koneksi internet berkecepatan tinggi, kamu mungkin memerlukan modem untuk menghubungkan ke layanan internet.

5. Firewall: Firewall dapat digunakan untuk melindungi jaringan dari ancaman keamanan dan mengatur lalu lintas internet.

6. Akses Point (AP): Jika Anda ingin memiliki jaringan Wi-Fi di laboratorium kamu.

31. Sebuah jaringan komputer terdiri dari 8⁶ router. Setiap router terhubung dengan 2⁵ perangkat lain dalam jaringan tersebut. Setiap perangkat memerlukan 2³ koneksi ke router. Berapa total jumlah koneksi yang dibutuhkan oleh seluruh perangkat dalam jaringan ini?A. 2¹⁴ koneksi jaringanB. 8¹⁴ koneksi jaringanC. 2²⁶ koneksi jaringanD. 8²⁶ koneksi jaringanE. 16²⁶ koneksi jaringanJawabannya pakai diketahui, ditanya, jawab, serta rumus ya kak. Makasih hehe​​


Diketahui:

- Jumlah router (R) = 8^6

- Setiap router terhubung dengan 2^5 perangkat

- Setiap perangkat memerlukan 2^3 koneksi ke router

Kita akan menggunakan rumus berikut:

Total jumlah koneksi = Jumlah router x (Jumlah perangkat per router x Jumlah koneksi per perangkat)

Jumlah perangkat per router = 2^5

Jumlah koneksi per perangkat = 2^3

Jadi,

Total jumlah koneksi = (8^6) x (2^5 x 2^3)

Kemudian, kita dapat menyederhanakan eksponen:

Total jumlah koneksi = (2^3)^6 x (2^5 x 2^3)

Total jumlah koneksi = 2^18 x 2^8

Total jumlah koneksi = 2^(18+8)

Total jumlah koneksi = 2^26

Jadi, total jumlah koneksi yang dibutuhkan oleh seluruh perangkat dalam jaringan ini adalah 2^26. Oleh karena itu, jawaban yang benar adalah:

C. 2^26 koneksi jaringan

Jawaban:

Jumlah router dalam jaringan komputer adalah 8⁶ = 8 x 8 x 8 x 8 x 8 x 8 = 262,144 router.

Setiap router terhubung dengan 2⁵ = 2 x 2 x 2 x 2 x 2 = 32 perangkat lain.

Jumlah perangkat dalam jaringan adalah 32 x 262,144 = 8,388,608 perangkat.

Setiap perangkat memerlukan 2³ = 2 x 2 x 2 = 8 koneksi ke router.

Jumlah total koneksi yang dibutuhkan oleh seluruh perangkat dalam jaringan ini adalah 8,388,608 perangkat x 8 koneksi = 67,109,056 koneksi.

Jadi, jawaban yang benar adalah A. 2¹⁴ koneksi jaringan.
