5. 5 contoh kalimat passive voice


Berikut rumus passive voice yang akan saya gunakan :
1). Simple present
Subject + to be (am/are/is) + V3 + by..
2). Present continuous
Subject + to be (am/are/is) + being + V3 + by...
3). Present perfect
Subject + have/has + been + V3 + by...
4). Simple past
Subject + to be (was/were) + V3 + by...
5). Simple future
Subject + will + be + V3 + by...

Contoh kalimat nya
1. Fried rice IS COOKED by mother.
2. Football IS BEING PLAYED by them.
3. The homework HAS BEEN DONE by us.
4. These books WERE BOUGHT by me yesterday.
5. She WILL BE VISITED by us tomorrow.

12. Jelaskan Passive voice dalam present continuous tense dan buat contoh beberapa yg menggunakan Positif negative Serta Interrogative​


Jawaban

Passive voice dalam present continuous tense digunakan ketika subjek tidak melakukan aksi dan menerima aksi dari objek.

Rumus passive voice dalam present continuous tense:

am/is/are + being + verb 3 (past participle)

Contoh kalimat:

Positif: The car is being washed by the mechanic. (Mekanik sedang mencuci mobil.)Negative: The dishes are not being washed by my brother. (Piring-piring tidak sedang dicuci oleh adik laki-laki saya.)Interrogative: Is the house being painted by the workers? (Apakah rumah sedang di cat oleh pekerja?)

Perlu diperhatikan bahwa dalam passive voice, objek menjadi subjek dan subjek menjadi objek. Jadi, dalam contoh kalimat di atas, mobil adalah objek dalam bentuk aktif, tetapi menjadi subjek dalam bentuk pasif.

Passive Voice adalah suatu bentuk di mana Subjek Kalimat menerima tindakan sedangkan Objeknya melakukan tindakan. Passive voice sendiri lebih menekankan kepada Objek kalimat yang terdampak dibandingkan Subjek kalimat yang merupakan pelaku.

Contoh 3 Passive Voice

⓵ Simple Present Tense

ᴥ Active:

(+) She studies English every night.(–) She doesn't study English every night.(?) Does she study English every night?

ᴥ Passive:

(+) English is studied by her every night.(–) English isn't studied by her every night. (?) Is English studied by her every night?

⓶ Simple Present Modal

ᴥ Active:

(+) I can cook chicken porridge.(–) I can't cook chicken porridge.(?) Can I cook chicken porridge?

ᴥ Passive:

(+) Chicken porridge can be cooked by me.(–) Chicken porridge can't be cooked by me. (?) Can chicken porridge be cooked by me?

⓷ Simple Present Perfect Tense

ᴥ Active:

(+) We have eaten chicken porridge.(–) We haven't eaten chicken porridge.(?) Have he eaten chicken porridge?

ᴥ Passive:

(+) Chicken porridge has been eaten by us.(–) Chicken porridge hasn't been eaten by us. (?) Has chicken porridge been eaten by us?

Berikut ini merupakan pola kalimat dari Active Voice dan Passive Voice:

Simple Present Tense

ᴥ Active Voice

(+) S + V1 (s/es) + O(–) S + do/does + not + V1 + O(?) Do/Does + S + V1 + O + ?

ᴥ Passive Voice

(+) S + is/am/are + V3 + by + agent(–) S + is/am/are + not + V3 + by + agent(?) Is/Am/Are + S + V3 + by + agent + ?

Simple Past Tense

ᴥ Active Voice

(+) S + V2 + O(–) S + did + not + V1 + O(?) Did + S + V1 + O + ?

ᴥ Passive Voice

(+) S + was/were + V3 + by + agent(–) S + was/were + not + V3 + by + agent(?) Was/Were + S + V3 + by + agent + ?

Simple Present Future Tense

ᴥ Active Voice

(+) S + will + V1 + O(–) S + will + not + V1 + O(?) Will + S + V1 + O + ?

ᴥ Passive Voice

(+) S + will + be + V3 + by + agent(–) S + will + not + be + V3 + by + agent(?) Will + S + be + V3 + by + agent + ?

Simple Past Future Tense

ᴥ Active Voice

(+) S + would + V1 + O(–) S + would + not + V1 + O(?) Would + S + V1 + O + ?

ᴥ Passive Voice

(+) S + would + be + V3 + by + agent(–) S + would + not + be + V3 + by + agent(?) Would + S + be + V3 + by + agent + ?

Simple Present Perfect Tense

ᴥ Active Voice

(+) S + has/have + V3 + O(–) S + has/have + not + V3 + O(?) Has/Have + S + V3 + O + ?

ᴥ Passive Voice

(+) S + has/have + been + V3 + by + agent(–) S + has/have + not + been + V3 + by + agent(?) Has/Have + S + been + V3 + by + agent + ?

Simple Past Perfect Tense

ᴥ Active Voice

(+) S + had + V3 + O(–) S + had + not + V3 + O(?) Had + S + V3 + O + ?

ᴥ Passive Voice

(+) S + had + been + V3 + by + agent(–) S + had + not + been + V3 + by + agent(?) Had + S + been + V3 + by + agent + ?

ᴥ Active Voice

(+) S + can/may + V1 + O(–) S + can/may + not + V1 + O(?) Can/May + S + V3 + O + ?

ᴥ Passive Voice

(+) S + can/may + be + V3 + by + agent(–) S + can/may + not + be + V3 + by + agent(?) Can/May + S + be + V3 + by + agent + ?

ᴥ Passive Voice

28. GERUND AND PASSIVE VOICE 1. Apakah yang dimaksud dengan Gerund? 2. Buatlah 3 contoh kalimat dengan menggunakan Gerund3. Apakah yang dimaksud dengan Passive Voice? Jelaskan dengan Rumus dari Passive Voice 4. Buatlah contoh 2 contoh kalimat passive voice​


Jawaban:

1. Gerund adalah bentuk kata kerja yang berfungsi sebagai kata benda (noun) dan dibentuk dengan menambahkan akhiran -ing pada kata kerja.

2. Contoh kalimat dengan menggunakan Gerund:

- Swimming is my favorite activity.

- I enjoy reading books in my spare time.

- Running helps me to stay healthy.

3. Passive Voice adalah suatu bentuk kalimat di mana subjek menerima tindakan yang dilakukan oleh objek. Rumus dari Passive Voice adalah:

Objek + to be (sesuai waktu) + kata kerja bentuk ketiga (V3) + oleh + subjek (optional)

4. Contoh kalimat Passive Voice:

- The cake was made by my mom.

- The movie was watched by us last night.
