4. contoh soal pilihan ganda persamaan dan pertidaksamaan nilai mutlak linear satu variabel


Contoh soal pilihan ganda persamaan dan pertidaksamaan nilai mutlak linear satu variabel

Pembahasan :

Definisi nilai mutlak :
|x| = x jika x ≥ 0
|x| = -x jika x < 0

|x| = √(x²)

Persamaan nilai mutlak (c > 0)
1) |ax + b| = c
=> ax + b = c atau ax + b = -c
2) |ax + b| = |cx + d|
=> (ax + b)² = (cx + d)²

Pertidaksamaan nilai mutlak (a> 0, c > 0)
1) |ax + b| < c
=> -c < ax + b < c
2) |ax + b| > c
=> ax + b < -c atau ax + b > c
3) |ax + b| < |cx + d|
=> (ax + b)² < (cx + d)²

Contoh soal :

1) Himpunan penyelesaian dari |2x - 7| = 5 adalah ...
a. {1}
b. {6}
C. {1, 6}
D. {-1}
E. {-1, 6}
Jawab :
|2x - 7| = 5
2x - 7 = 5 atau 2x - 7 = -5
2x = 12 atau 2x = 2
x = 6 atau x = 1
HP = {1, 6}
Jawaban C

2) Himpunan penyelesaian dari persamaan |2x + 3| = |x + 6| adalah ...
A. {3}
B. {0}
C. {0, 3}
D. {-3, 3}
E. {-3, 0}
Jawab :
|2x + 3| = |x + 6|
(2x + 3)² = (x + 6)²
4x² + 12x + 9 = x² + 12x + 36
3x² - 27 = 0
x² - 9 = 0
(x + 3)(x - 3) = 0
x = -3 atau x = 3
HP = {-3, 3}
Jawaban D

E. -2 ≤ x ≤ 7
Jawab :

4) Himpunan penyelesaian dari |3x - 2| > |x + 3| adalah ....
A. x < 1/4 atau x > 5/2
B. x < -1/4 atau x > 5/2
C. x < -5/2 atau x > 1/4
D. -5/2 < x < 1/4
E. -1/4 < x < 5/2
Jawab :
|3x - 2| > |x + 3|
(3x - 2)² > (x + 3)²
9x² - 12x + 4 > x² + 6x + 9
8x² - 18x - 5 > 0
(4x + 1)(2x - 5) > 0
x = -1/4 atau x = 5/2
Garis bilangan :
+++++ (-1/4) ------ (5/2) ++++
x < -1/4 atau x > 5/2
Jawaban B

15. ekonomi soal 22 pilihan ganda​


Dengan kurs beli sebesar Rp 9.250 dan kurs jual sebesar Rp 9.400, Laura yang merupakan turis dari Amerika membawa uang US$ 5.000 lalu membelanjaka uangnya sebesar Rp 35.000.000. Dengan informasi tersebut, sisa dari uang Laura adalah US$1297,3.

Pembahasan

Pilihan jawaban yang ada pada soal tidak ada yang benar karena dengan informasi yang ada pada soal, maka hal pertama yang akan dilakukan oleh Lauran adalah menukarkan US$ 5.000 yang ia bawa dari Amerika ke mata uang Indonesia. Sehingga didapatkan 5.000 x Rp 9.400 = Rp 47.000.000. Ketika di Indonesia, Laura menjajakan uangnya sebesar Rp 35.000.000 sehingga sisa uang Laura adalah Rp 12.000.000. Lalu, Laura akan menukarkan uangnya kembali ke US$ sehingga didapatkan 12.000.000 / 9.250 = US$ 1297.29 atau US$ 1297.3.

Valuta asing adalah salah satu alat bayar yang bisa kita gunakan ketika melaksanakan perdagangan internasional. Selain itu, kita juga harus bisa menyesuaikan mata uang kita ketika kita berada di negara lain yang menggunakan mata uang berbeda. Kurs beli merujuk pada rasio ketika kita membeli uang tersebut sedangkan kurs jual merujuk pada rasio yang diberikan ketika kita menjual mata uang tersebut.

23. contoh soal non linear


(2/x) + (3/y) = 12 dan (3/x) – (1/y) = 7
Misalkan 1/x = a dan 1/y = b, maka:
(2/x) + (3/y) = 12 => 2a + 3b = 12
(3/x) – (1/y) = 7 => 3a – b = 7
Kita gunakan cara cepat, yakni:
=> b = (2.7 – 3.12)/(2.( –1) – 3.3)
=> b = (14 – 36)/( –2 – 9)
=> b = –22/–11
=> b = 2
Substitusi nilai b = 2 ke persamaan 3a – b = 7, sehingga diperoleh:
=> 3a – b = 7
=> 3a – 2 = 7
=> 3a = 7 + 2
=> 3a = 9
=> a = 3
Kembalikan nilai a dan b ke pemisalan semula, yakni:
1/x = a
=> 1/x = 3
=> x = 1/3

1/y = b
=> 1/y = 2
=> y = 1/2
Jadi, penyelesaian persamaan tersebut adalah x = 1/3 dan y = 1/2

26. buatlah 2 contoh soal pilihan ganda, 1 finite clause, dan 1 non finite clause


Untuk membuat 2 (dua) contoh soal mengenai finite clause dan non finite clause, penting untuk memahami kedua jenis klausa tersebut. Berikut akan dibahas definisi finite clause dan non finite clause. Setelah pembahasan singkat, akan saya berikan dua contoh soal pilihan ganda dengan finite clause dan non finite clause.

Pembahasan

Finite clause dapat dilihat dari kata kerja (verb) yang ada di dalam kalimat. Kata kerja yang digunakan akan berubah sesuai dengan grammar kalimat.

Contoh kalimat dengan finite clause:

- She felt hungry 3 hours ago.

Kata kerja felt adalah kata kerja lampau. Kata kerja ini dapat berubah menjadi feels ketika digunakan untuk kalimat present tense, yaitu: She feels sick every morning.

Contoh lainnya adalah:

- I sprained my ankle when I was playing tennis yesterday.

I sprained my ankle dan I was playing tennis yesterday adalah dua kalimat yang dapat berdiri sendiri. Keduanya dikenal sebagai finite clause.

Non finite clause adalah kalimat yang tidak menggunakan kata kerja sesuai grammar dan tidak dapat berdiri sendiri. Non finite clause biasanya bukan kalimat utama.

Contoh non finite clause:

- I am excited to see you.

To see you adalah non finite clause.

- I don't want to feel tired.

To feel tired adalah non finite clause.

Non finite clause umumnya tidak memiliki subjek.
