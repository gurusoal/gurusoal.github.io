8. If You _________ (not behave), I _________ (throw) you out. Type l : Type ll : Type lll : gmn yah ?? soalnya belom di jelasin​


Jawaban:

If You _________ (not behave), I _________ (throw) you out.

Type l : if you don't behave, I will throw you out.

Type ll : if you didn't behave, I would throw you out.

Type lll : if you hadn't behaved, I would have thrown you out.

Penjelasan:

Ini nama nya Conditional sentence atau kalimat pengandaian. Ada 3 tipe.

Tipe 1 pakai present tense

Rumus: If + S + Verb1 + O, S + will + Verb0 + O

Tipe 2 pakai past tense

Rumus: If + S + Verb2 + O, S + would + Verb0 + O

Tipe 3 pakai past perfect tense

Rumus: If + S + had + Verb3 + O, S + would + have + Verb3 + O

Kalimat bisa di balik ya, pakai will dulu baru if.

Note:

* Verb0 maksud nya kata kerja yang tidak ada penambahan atau perubahan apapun.

Kalau masi belum mengerti, bisa tanya guru bahasa Inggris atau googling, ada banyak contoh soal dan penjelasan nya juga.

10. Contoh soal kalimat conditional sentence type 2


If my father were rich, I would live in a big villa with swimming polo inside.

(Jika ayah saya kaya, saya akan tinggal di sebuah vila besar dengan kolam renang di dalamnya.)

Keterangan: Dalam kalimat pengandaian ini, if my father was rich merupakan dependent clause yang tidak akan memiliki makna jika berdiri sendiri, sedangkan i would live in a big villa with swimming polo inside merupakan independent clause yang tetap memiliki makna meskipun tidak diikuti dengan kalimat di depan.


11. apa yang dimaksud DOC


Dalam teknologi komputasi, DOC atau doc (singkatan dari 'document' ) adalah sebuah ekstensi berkas untuk aplikasi pengolah kata yang terdapat di Microsoft Word Binary File Format.[4] Sejarah mengatakan bahwa ekstensi tersebut pernah digunakan pada teks biasa, khususnya program atau perangkat keras komputer di berbagai sistem operasi. Pada 1980an, WordPerfect menggunakan DOC sebagai format bawaan mereka. Kemudian, pada dasawarsa 1990an, Microsoftmemilih untuk menggunakan ekstensi DOC untuk format bawaaan Microsoft Word.

15. cari 2 soal kalimat di bikin type1 type 2 real type 3 real


Kelas: SMA
Pelajaran: English
Kategori: conditional
Pembahasan: 

Type 1 (possible event)
- If she calls me, I'll ask her.
- If she doesn't call me, I won't call her first.

Type 2 (imagination)
- If the alien attacked earth, I'd try to get to know it.
(the real: alien doesn't attack earth, so I won't try to get to know it)
- if North Korea started the war against USA, there would be a World War III. (the real: North Korea doesn't start the war yet, so there is no WW III.)

Type 3 (regrets)
- I would have sent it earlier, if I had known about it. 
(the real: I didn't know about it, so I wouldn't send it earlier)
- The murderer would have been caught, if the cops hadn't been fooled by him. (the real: the murderer wasn't caught because the cops were fooled by him)

17. adakah perbedaan antara soal-soal type AKM dengan soal HOTs? Jelaskan!


• Soal tipe AKM

Soal AKM terdiri dari soal literasi dan numerasi

1. Soal literasi antara lain disajikan teks bacaan yang cukup panjang sebagai stimulus kemudian ditanyakan tentang

a. gagasan pokok teks

b. simpulan teks

c. persamaan isi beberapa teks

d. pernyataan yang sesuai/tidak sesuai dengan isi teks

e. pernyataan sesuai atau tidak dan diminta menuliskan alasan

2. Soal Numerasi

Disajikan data berupa angka dalam bentuk tabel dan grafik

ditanyakan

a. isi data

b. data yang sesuai/tidak sesuai

c. pernyataan disertai alasan jawaban

3. Soal Survey Karakter

Disajikan sebuah ilustrasi peristiwa tokoh menghadapi situasi tertentu, kemudian disajikan pilihan jawaban tindakan/sikap tokoh menghadapi situasi tertentu. Berdasarkan pilihan tindakan/sikap yang dipilih, dapat ditentukan karakter seseorang

• Soal tipe HOTS

Soal HOTS adalah model evaluasi pendidikan yang menguji kemampuan berpikir tingkat tinggi. Soal HOTS akan mengasah logika, pola pikir kritis, dan kreativitas siswa
