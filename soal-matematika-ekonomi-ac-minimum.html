6. Jika diketahui fungsi biaya total pada perusahaan adalah TC=0,4Q^2+500Q+16000 a.carilah fungsi biaya rata-rata (AC)? b.berapakah jumlah produk yang di hasilkan agar biaya rata-rata minimum? c.berapa nilai biaya rata-rata minimum tersebut?


TC=0,4Q^2+500Q+16000
a. AC = 0.4Q + 500 + 16000/Q

b. AC minimum :
AC' = 0.4 + -16000/Q^2 = 0
0.4Q^2 - 16000 = 0
4Q^2 - 160000 = 0
(2Q + 400)(2Q - 400) = 0
Q = 200 (karena jumlah produk tidak mungkin minus)

c. Ketika Q = 200
AC = 0.4(200) + 500 + 16000/200
AC = 80 + 500 + 80
AC = 660

37. Fungsi biaya rata-rata berbentuk parabola AC=0,25Q^2-25Q+225. Tentukan : a. Bentuk fungsi TC dan MC b. Berapa Q ketika AC minimum c. Buktikan ketika AC minimum, nilai AC=MC mohon bantuannya berupa cara penyelesaiannya. terimakasih


a) TC = AC×Q
TC = (0,25Q² - 25Q + 225) × Q
TC = 0,25Q³ - 25Q² + 225Q
MC = TC'
MC = 0,75Q² - 50Q + 225
b) turunan AC = 0
0,5Q - 25 = 0
0,5Q = 25
Q = 25/0,5 = 50
c) minimum saat Q = 50
AC = MC
0,25Q² - 25Q + 225 = 0,75Q² - 50Q + 225
0 = 0,75Q²-0,25Q²-50Q+25Q+225-225
0 = 0,50Q² - 25Q
0,50Q² - 25Q = 0
Q(0,50Q - 25) = 0
Q = 0
0,50Q - 25 = 0
0,50Q = 25
Q = 25/0,50
Q = 50 (terbukti saat biaya min Q = 50)
