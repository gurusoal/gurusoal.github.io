9. bagaimanakah merubah slide presentasi di Microsoft powerpoint ​


Jawaban:

Penjelasan:

Untuk mengubah slide presentasi di Microsoft PowerPoint, Anda dapat mengikuti langkah-langkah berikut:

1. Buka presentasi PowerPoint yang ingin Anda ubah slide-nya.

2. Di bagian bawah tampilan PowerPoint, Anda akan melihat daftar slide yang ada dalam presentasi. Klik pada slide yang ingin Anda ubah.

3. Setelah mengklik slide yang diinginkan, Anda dapat melakukan berbagai perubahan pada slide tersebut, seperti mengedit teks, menambahkan gambar, memodifikasi tata letak, dan sebagainya.

4. Untuk mengubah teks pada slide, cukup klik di dalam teks yang ingin diubah, kemudian mulai mengetik atau menghapus teks yang ada.

5. Jika Anda ingin menambahkan gambar, klik pada tab "Insert" di menu atas PowerPoint, lalu pilih opsi "Pictures" atau "Online Pictures" untuk menyisipkan gambar dari komputer atau dari sumber gambar online.

6. Untuk mengubah tata letak slide, klik pada tab "Home" di menu atas PowerPoint, lalu pilih opsi "Layout" untuk memilih tata letak slide yang diinginkan. Anda dapat memilih tata letak dengan judul dan teks, tata letak dengan gambar besar, atau tata letak khusus lainnya sesuai kebutuhan.

7. Selain itu, Anda juga dapat menambahkan animasi, transisi, grafik, tabel, dan elemen-elemen visual lainnya sesuai keinginan Anda. Untuk melakukannya, gunakan opsi yang tersedia di menu atas PowerPoint, seperti "Animations", "Transitions", atau "Insert" untuk menyisipkan elemen-elemen tersebut.

8. Setelah Anda melakukan perubahan pada slide, pastikan untuk menyimpan presentasi PowerPoint Anda agar perubahan yang Anda buat tersimpan.

Dengan langkah-langkah di atas, Anda dapat merubah dan mengedit slide presentasi dengan mudah menggunakan Microsoft PowerPoint.

11. Cara mengirimkan presentasi melalui Microsoft PowerPoint melalui.....


Pakai E-mail caranya:

Pilih File > Bagikan > Email

Di bawah Email, klik salah satu hal berikut:

Kirim sebagai Lampiran untuk melampirkan presentasi Anda ke dalam pesan email.

Kirim Link untuk membuat email yang berisi link ke presentasi Anda.

CATATAN: Sebelum Anda Mengirim Link, Anda harus menyimpan file Anda ke lokasi bersama, seperti situs web atau pustaka dokumen di mana penerima email memiliki izin untuk mengakses lokasi bersama tersebut.

Kirim sebagai PDF untuk menyimpan presentasi Anda sebagai file Portable Document Format (.pdf), lalu lampirkan file PDF tersebut ke dalam pesan email.

Kirim sebagai XPS untuk menyimpan presentasi Anda sebagai file .xps, lalu lampirkan file tersebut ke dalam pesan email.

Kirim sebagai Faks Internet untuk mengirim presentasi Anda sebagai faks tanpa menggunakan mesin faks. Opsi ini memerlukan Anda untuk mendaftarkan diri terlebih dahulu ke penyedia layanan faks. Jika Anda belum mendaftar ke penyedia layanan faks, klik Kirim sebagai Faks Internet, dan Anda akan diarahkan langsung ke situs web di mana Anda dapat memilih penyedia layanan.
