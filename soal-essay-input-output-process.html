26. Tahapan proses kerja komputer yang benar adalah a Input-output-proses b proses-input-output c input-proses-output d proses-output-input e output-input-output ​


Jawaban :

c input-proses-output

pembahasan:

Tahapan proses kerja komputer yang benar

1. Pemasukan (Input)

Tahap pemasukan (input) merupakan tahap awal dari proses pengolahan yang terjadi pada sistem komputer. Tahap ini berupa pemasukan data mentah ke dalam sistem komputer melalui input device. Contoh dari peralatan masukan (input device) diantaranya, keyboard, mouse, scanner, dll. Alat-alat inilah yang digunakan untuk memasukan data yang hendak diolah. Seperti hal nya keyboard, difungsikan untuk memasukan huruf, angka, maupun simbol-simbol lainnya ke dalam komputer yang selanjutnya akan diproses

2. Pemrosesan (Process)

Pada tahap ini, data yang telah dimasukan melalui peralatan input tadi akan diproses. Tahap proses ini dilakukan oleh processing device yaitu CPU. Yang mana CPU ini dapat melakukan fungsi perhitungan dan logika untuk perbandingan (ALU) dan juga mengontrol (CU). Pada tahap ini, data yang masih mentah tadi diproses sedemikian rupa sehingga data tersebut siap dicetak menjadi informasi yang lebih bermanfaat.

3. Pengeluaran (Output)

Pada tahap ini, data yang tadinya telah dimasukan melalui peralatan input, kemudian diproses oleh CPU akan bisa dicetak apabila sudah siap. Pencetakan ini bisa berupa hardcopy dan juga softcopy. Hard copy berarti menggunakan media fisik seperti kertas ataupun yang lainnya. Softcopy berarti menampilkan gambar visual melalui monitor ataupun projektor. Yang termasuk dalam peralatan output disini adalah, monitor, projector, printer, dll.

38. Berikut yang bukan merupakan ciri-ciri algoritma adalah....A . FinitenessB . DefinitenessC. InputD. OutputE. Process​


Yang bukan merupakan ciri-ciri algoritma adalah input (pilihan C). Input merupakan data yang diberikan ke algoritma untuk diproses. Input dapat berupa angka, teks, gambar, atau data lainnya.

Pembahasan

Input tidak termasuk dalam ciri-ciri algoritma karena input dapat bervariasi tergantung pada masalah yang ingin diselesaikan oleh algoritma tersebut.

Terangkan alur suatu projek bermula dari input device sampai ke output device (disusun atau ditulis dalam format prosedur text)​


1. Input Device Mengirimkan informasi

Ke Sistem Komputer Untuk Diproses,

Dan Output device Mereproduksi Atau Menampilkan Hasil Pemrosesan Tersebut. Input Device Hanya Mengizinkan input Data Ke Komputer Dan Output device Hanya Menerima output Data Dari Perangkat lain. Process Device Adalah Semua Perangkat Di Dalam Komputer yang Menafsirkan Dan Memanipulasi Data yang Masuk Selama Tahap Ini.

2. Contoh Input Device yaitu: Keyboard, Mouse, Microfon, Dan Webcam.

Contoh Output Device yaitu: Monitor, Proyektor, Dan Speaker.

Contoh Process Device yaitu: CPU, Memory Dan Motherboard.
