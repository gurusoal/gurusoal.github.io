1. TOLONG PAKE BAHASA SUNDA YA NOMOR 3 ITU CONTOH SOAL ESSAY UKK


Kategori Soal : Bahasa Sunda - Dongeng
Kelas : VII (1 SMP)
Pembahasan :

Calung mangrupa pakakas musik ti daerah Sunda anu bentukna siga angklung. Tapi dina cara maenkeunnana, calung beda jeung angklung. Calung mah dimaenkeunnana ku ditakol, sedengkeun angklung mah dimaenkeunnana digoyang. Salian ti eta, dina tangga nada anu digunakeun, calung oge beda jeung angklung. Calung ngagunakeun tangga nada pentatonis (da-mi-na-ti-la), sedengkeun angklung ngagunakeun tangga nada diatonis (do-re-mi-fa-sol-la-si).

Seni calung biasana mangrupa seni pintonan anu dimaenkeun ku sawatara urang, saperti opat atawa lima urang anu boga sipat hiburan sarta mere informasi atawa panyuluhan. Seni calung biasana dibawakeun bari ngabodor. Kusabab seni calung dibawakeun bari ngabodor, eta teh matak dipikaresep ku anu lalajo. Jadi cara mintonkeunnana calung teh beda jeung angklung. Umpama angklung dipintonkeun teu beda saperti pakakas musik lianna, calung mah dipintonkeun bari ngabodor.

Penjelasan panjang:

15. MOD ialah salah 1 fungsi dalam Ms. Excel untuk mencari sisa hasil bagi data angka. Format penulisan fungsi MOD yaitu:

=MOD(angka;dibagi_berapa)

Berdasarkan soal di atas, fungsi MOD digabung dg fungsi IF dan sesuai kaidah penulisan di atas, jawaban yg tepat adalah C

Pembacaan fungsinya menjadi:

"Jika angka dalam sel C2 dibagi 2 sama dengan 0, maka tampilkan B, jika tidak sama dengan 0, maka tampilkan A"

16. Penelusuran hasil fungsi logika dalam sel A2 yaitu:

=OR(AND(A1:B1);AND(B1:C1))

=OR(FALSE;FALSE)

Hasilnya FALSE

Penelusuran hasil fungsi logika dalam sel A3 yaitu:

=AND(OR(A1:B1);OR(B1:C1))

=AND(TRUE;FALSE)

Hasilnya FALSE

Penelusuran hasil fungsi logika dalam sel A4 yaitu:

=OR(AND(A1:B1);OR(A1:C1))

=OR(FALSE;TRUE)

Hasilnya TRUE

Perlu diingat bahwa:

OR akan menghasilkan TRUE bila melibatkan salah 1 pernyataan TRUE

OR akan menghasilkan FALSE bila semua pernyataan adalah FALSE

AND akan menghasilkan TRUE bila semua pernyataan adalah TRUE

5. Pak Dani sedang berusaha merintis usaha kuliner. karena tidak memiliki cukup uang, maka ia meminjam uang ke Bank Lampung sebesar Rp. 40.000.000. ternyata modal tersebut masihlah kurang maka ia meminjam lagi Rp. 13.000.000. sebulan kemudian pak Dani mampu membayar hutangnya sebesar Rp. 50.000.000. berapakah sisa hutang pak Dani?

PEMBAHASANNYA :
1. Modal awal = Rp.6.500.000     keuntungan hari ke 1 = Rp.3.500.000
    kerugian hari ke 2 = Rp.1.500.000

    Rp.3.500.000 - Rp.1.500.000 = Rp. 2.000.000

jadi untuk mengembalikan modal pak Abdul membutuhkan = Rp. 6.500.000 - Rp. 2000.000 = Rp. 4.500.000

3. X = 79 : Y = 64

   x = 91 : Y = 64

agar menang 1 poin maka Y membutuhkan : 91 + 1 - 64 = 92 - 64 = 28 Poin

    diturunkan sebanyak 5 kotak = 5 x 24 = 120 buah apel
    maka sisa apel di dalam pickup = 288 - 120 = 168 buah apel
